Deteksi Phishing URL berbasis Machine Learning

Proyek Case-Based Learning — mata kuliah Desain Analisis Sains Data (DASD) terintegrasi Sistem Informasi Analitik Bisnis (SIAB), Universitas Aisyiyah Yogyakarta.

Apa itu Deteksi Phishing URL?

Phishing adalah teknik penipuan di mana pelaku membuat website palsu yang meniru tampilan situs resmi (bank, e-commerce, media sosial, dsb) untuk mencuri data sensitif korban seperti username, password, atau data kartu kredit.

Aplikasi ini menggunakan model Machine Learning (Random Forest) yang dilatih pada dataset PhiUSIIL Phishing URL Dataset untuk memprediksi apakah sebuah URL kemungkinan besar Phishing atau Legitimate (asli/aman), berdasarkan 15 karakteristik teknis (hasil seleksi top feature importance) dari URL dan halaman webnya.

Kenapa ini penting?

Serangan phishing terus meningkat dan sering menjadi titik masuk pencurian data, penipuan finansial, hingga ransomware. Deteksi dini berbasis machine learning membantu tim keamanan TI dan pengguna umum mengenali URL berisiko sebelum data sensitif diberikan ke situs yang salah.

Cara Kerja Sistem

  1. Pengguna memasukkan URL pada halaman Cek URL.
  2. Sistem mengekstrak fitur dari URL (panjang, HTTPS, subdomain, dll) dan dari HTML halamannya (gambar, script, iframe, form, dll).
  3. Fitur dimasukkan ke model Random Forest yang dilatih pada ±234 ribu data URL berlabel.
  4. Model mengembalikan prediksi beserta skor keyakinan.
